# üí´ About Me

PhD student at The Scripps Research Institute in the Yates Lab, developing Python tools for computational proteomics workflows. I maintain a suite of open-source packages and web applications for mass spectrometry data analysis. Currently setting up a server at Scripps to host production apps and services.

**Active projects**: MS1 feature finder, real-time mass spectrometry applications, machine learning modules for peptide property prediction, and MS1/MS2 search engines.

## üì¶ Core Packages

**Proteomics Analysis**
- **[Peptacular](https://github.com/pgarrett-scripps/peptacular)** - Proforma 2.1 compliant utility package for amino acid sequences (similar to pyteomics)
- **[FastaFrames](https://github.com/pgarrett-scripps/FastaFrames)** - Convert between FASTA files and Pandas DataFrames
- **[MsDecon](https://github.com/pgarrett-scripps/MsDecon)** - Simple-ish MS deconvolution using rustworkx graph algorithms
- **[tdfpy](https://github.com/pgarrett-scripps/tdfpy)** - Parse Bruker timsTOF data with centroiding and noise filtering
- **[MsMsCompression](https://github.com/pgarrett-scripps/MsMsCompression)** - URL-compatible compression algorithms for mass spectra

**Data Processing**
- **[SagePeptideAmbiguityAnnotator](https://github.com/pgarrett-scripps/SagePeptideAmbiguityAnnotator)** - Annotate Sage output with missing fragments and unknown mass shifts
- **[msconvert-cli](https://github.com/pgarrett-scripps/msconvert-cli)** - Command-line wrapper for msconvert Docker image
