# üí´ About Me

PhD student at The Scripps Research Institute in the Yates Lab, developing Python tools for computational proteomics workflows. I maintain a suite of open-source packages and web applications for mass spectrometry data analysis.

**üî¨ Streamlit Tools Homepage**: https://proteomics-tools.streamlit.app/

## üì¶ Core Packages

**Proteomics Analysis**
- **[Peptacular](https://github.com/pgarrett-scripps/peptacular)** - Proforma 2.1 compliant utility package for amino acid sequences (similar to pyteomics)
- **[FastaFrames](https://github.com/pgarrett-scripps/FastaFrames)** - Convert between FASTA files and Pandas DataFrames
- **[MsDecon](https://github.com/pgarrett-scripps/MsDecon)** - MS deconvolution using rustworkx graph algorithms
- **[tdfpy](https://github.com/pgarrett-scripps/tdfpy)** - Parse Bruker timsTOF data with centroiding and noise filtering
- **[MsMsCompression](https://github.com/pgarrett-scripps/MsMsCompression)** - Web-compatible compression algorithms for mass spectra

**Data Processing**
- **[SagePeptideAmbiguityAnnotator](https://github.com/pgarrett-scripps/SagePeptideAmbiguityAnnotator)** - Annotate Sage output with missing fragments and unknown mass shifts (e.g., `PEP(?TID)[+500]E`)
- **[msconvert-cli](https://github.com/pgarrett-scripps/msconvert-cli)** - Command-line wrapper for msconvert Docker image

## üåê Web Applications

- **[ProteinCleaverStreamlitApp](https://github.com/pgarrett-scripps/ProteinCleaverStreamlitApp)** - In silico protein digestion and enzyme site visualization
- **[PeptideFragmenterStreamlitApp](https://github.com/pgarrett-scripps/PeptideFragmenterStreamlitApp))** - Interactive peptide fragment ion viewer
- **[StreamlitSpectrumViewer](https://github.com/pgarrett-scripps/StreamlitSpectrumViewer)** - Annotated mass spectrum visualization
- **[CSV_Viewer_Streamlit](https://github.com/pgarrett-scripps/CSV_Viewer_Streamlit)** - Universal file viewer for common formats

## üé® Streamlit Extensions

- **[Streamlit_Notify](https://github.com/pgarrett-scripps/Streamlit_Notify)** - Status notification queue system (part of Streamlit Extras)
- **[streamlit-permalink](https://github.com/pgarrett-scripps/streamlit-permalink)** - Stateful widget URL parameters (forked)
